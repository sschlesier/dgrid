# DGrid

A modern MongoDB GUI for macOS.

## Install (macOS)

```bash
brew install --cask sschlesier/dgrid/dgrid
```

Or download the latest `.dmg` from [Releases](https://github.com/sschlesier/dgrid/releases).

## Development

```bash
pnpm install
pnpm dev
```

Open http://localhost:5173 in your browser.
